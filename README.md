# dream-bakes
Full-stack bakery management system built with Spring Boot and React.

- View cakes, pastries, donuts, bagels, specials of the day
- Place custom cake design requests
- Browse daily specials

Admin users can:
- View custom orders
- Approve or reject orders
- Add daily specials
- Manage products
